# Stella

> Данный проект был изначально написан для ТГ-комьюнити игры Warplane Inc Online, в будущем планирую полностью переписать проект для большей универсальности.

Бот-модератор для Telegram + веб-панель управления.

## Возможности

- Модерация чатов: мут, бан, кик с указанием срока и причины
- Испытательный срок для рецидивистов
- Система апелляций
- Веб-панель: дашборд, управление пользователями, настройки, журнал действий
- Интеграция с DeepSeek для ИИ-ассистента

## Запуск

```bash
cp .env.example .env   # заполнить BOT_TOKEN, ADMIN_PASSWORD
pip install -r requirements.txt
python run.py all      # бот + панель
```

Docker: `docker-compose up -d`

## Структура

```
core/        — конфигурация, модели, настройки
web/routes/  — Flask-панель (9 модулей)
handlers/    — обработчики бота (11 модулей)
tests/       — pytest
deploy/      — systemd, nginx
```

## Тесты

```bash
pip install -r requirements-dev.txt
pytest tests/ -v
ruff check .
```

Проект в стадии активной разработки. В планах — частичный переход на Go для критичных по производительности компонентов и переработка архитектуры под мультитенантность.
